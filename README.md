
# ATM Chatbot

## Overview
This project aims to create an ATM chatbot using Keras RNN for conversational interface and voice integration in Hindi, Tamil, and English.

## Features
- **Authentication**: Users can authenticate using PIN, fingerprint, or face recognition.
- **Main Menu**: Options include balance inquiry, cash withdrawal, deposit, transfer funds, change PIN, and help.
- **Voice Integration**: Seamless voice output in multiple languages.
- **Balance Inquiry**: Users can check their account balance.
- **Cash Withdrawal**: Withdraw cash securely.
- **Deposit**: Deposit cash or checks conveniently.
- **Transfer Funds**: Transfer funds to other accounts.
- **Change PIN**: Change PIN securely.
- **Help**: Provides assistance for various tasks.

## Implementation
The chatbot is developed using Keras RNN with Python. Voice integration is achieved through text-to-speech (TTS) for Hindi, Tamil, and English languages.
